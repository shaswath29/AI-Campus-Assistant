# AI Campus Assistant

## Objective
Build an intelligent assistant to help students navigate campus life.

## Level 1: Basic AI Assistant
Simple Q&A using a pre-trained LLM.

## Level 2: Knowledge-Aware Assistant (RAG)
Processes campus documents like FAQs, rules, and schedules.

## Level 3: Agentic Campus Assistant
Handles multi-step queries with memory and tool usage.

## Tech Stack
- Python
- Streamlit
- GitHub
